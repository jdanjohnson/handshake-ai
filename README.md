# Handshake Legal

Handshake Legal is a platform that simplifies the process of creating, sharing, and digitally signing agreements.

## Core Features

- **Offer Creation**: Allows users to create new offers with specified terms.
- **Offer Sharing**: Generates a unique and secure URL for each offer to be shared.
- **Offer Review**: Enables the receiving party to securely view and review the offer's terms.
- **Digital Handshake**: Provides a simple and clear way for the receiving party to accept the offer.
- **Agreement Generation**: Automatically generates a legally enforceable agreement in PDF format upon acceptance.
- **AI Completeness**: Utilizes generative AI to ensure that the agreement includes all necessary details, providing prompts and suggestions.
- **Archival**: Provides an archival system for all parties to access previous offers and agreements.
